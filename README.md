# How to play :game_die:
* กดปุ่มลูกศรบนคีย์บอร์ดเพื่อควบคุมทิศทางของงู
* หากเก็บของขนาดของงูจะเพิ่มขึ้น 1
* หากชนตัวเองจะตาย
* สามารถทะลุกรอบ/กําแพงได้
